マスターページ（編集用）

会員TOPの「年間カレンダー」「直近の講座動画」「直近の活動告知」は、data.js を編集すると更新されます。

← 会員TOPに戻る

編集の流れ

  1. data.js をテキストエディタで開く。
  2. calendar（年間カレンダー）・youtube（講座動画）・announcements（活動告知）の中身を編集する。
  3. ファイルを保存し、会員TOP（index.html）を再読み込みして表示を確認する。

1. 年間カレンダー（日付・会場の変更）

data.js の calendar 配列を編集します。各項目は month（月）・event（行事名）・date（日付）・venue（会場）です。

2. 直近の講座動画（YouTube）

data.js の youtube 配列を編集します。各項目は videoId（YouTubeの動画ID）・title（表示タイトル）・published（公開日表示）です。配列の先頭が会員TOPで一番上に表示されます。

3. 直近の活動告知（追加・変更）

data.js の announcements 配列を編集します。各項目は date（日付）・title（タイトル）・body（本文）です。配列の先頭が会員TOPで一番上に表示されます。

data.js の編集例

日付・会場が決まったら、該当する行の date や venue を書き換えます。

// 例：第1講の日付が 5/10、会場が 〇〇会館 の場合
{ month: "5月", event: "第1講", date: "5/10", venue: "〇〇会館" },

動画を1本追加する場合は、youtube 配列にオブジェクトを追加します。videoId は YouTube の URL ?v=xxxxx の xxxxx の部分です。

{ videoId: "動画ID11文字", title: "第3講 講座タイトル", published: "2026/07/15" },

告知を1件追加する場合は、announcements 配列の先頭にオブジェクトを追加します。

{ date: "2026/04/01", title: "新着のお知らせ", body: "本文をここに書きます。" },
