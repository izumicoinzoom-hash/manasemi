# 年間カレンダー（日付・会場）の更新方法

会員TOPの「年間カレンダー」セクションは、**日時・会場が決まったものから随時、日付・会場を埋めていく**想定です。

## 更新するファイル

- **`member/index.html`** の「年間カレンダー」セクション内の **&lt;tbody&gt;** にある表（&lt;table&gt;）を編集します。

## 表の列構成

| 列 | 内容 |
|----|------|
| 月 | 3月、4月、5月… または — |
| 主な行事 | オープンゼミ、第1講、志心の会 全体例会 など |
| 日付 | 決まった日付（例：5/10）または — |
| 会場 | 決まった会場名・住所など または — |

## 編集の仕方

1. `member/index.html` をテキストエディタで開く。
2. 「年間カレンダー」または `id="calendar"` で検索し、該当の **&lt;table&gt;** を探す。
3. 各行事の行（&lt;tr&gt;）の **3列目「日付」**・**4列目「会場」** の &lt;td&gt; 内の「—」を、決まった内容に書き換える。

### 日付の書き方の例

- **決まった日付**: `3/15` や `3月15日`、`2026年3月15日` など、運用に合わせて統一するとよいです。
- **未定のまま**: `—` または `未定` のままにする。

### 会場の書き方の例

- **決まった会場**: `〇〇会館 3階`、`〇〇市民センター` など。必要に応じて住所や備考を短く追記しても可。
- **未定のまま**: `—` または `未定` のままにする。

### 例（第1講の日付が 5/10、会場が 〇〇会館 に決まった場合）

**変更前:**
```html
<tr class="border-b border-gray-200"><td class="py-3 px-4 font-medium text-brand-navy">5月</td><td class="py-3 px-4">第1講</td><td class="py-3 px-4">—</td><td class="py-3 px-4">—</td></tr>
```

**変更後:**
```html
<tr class="border-b border-gray-200"><td class="py-3 px-4 font-medium text-brand-navy">5月</td><td class="py-3 px-4">第1講</td><td class="py-3 px-4">5/10</td><td class="py-3 px-4">〇〇会館</td></tr>
```

## 行事の追加・削除

- **行を増やす**: 同じ &lt;tr&gt;～&lt;/tr&gt; をコピーし、月・行事名・日付・会場を書き換える。（1行につき &lt;td&gt; は4つ）
- **行を減らす**: 不要な &lt;tr&gt;～&lt;/tr&gt; を削除する。

更新後、ファイルを保存し、Git でコミット・プッシュするとサイトに反映されます。
