# 年間カレンダー・活動告知の更新方法（マスターデータ方式）

会員TOPの**年間カレンダー**・**直近の講座動画（YouTube）**・**直近の活動告知**は、**data.js** を編集すると更新されます。HTMLを直接触る必要はありません。

## マスターページの利用（編集・保存が簡単）

- **master.html** にアクセスすると、**コード入力**が求められます。正しいコードを入力すると編集画面に入れます。
- コードは **master-config.js** の `MANASEMI_MASTER_CODE` で設定・変更できます。運用時に必ず変更してください。
- 編集画面では、カレンダー・講座動画・活動告知を**フォームで直接編集**できます。「**data.js を保存（ダウンロード）**」ボタンで、編集内容が反映された data.js がダウンロードされます。
- ダウンロードした **data.js** を `member/data.js` の場所に上書きし、Git でコミット・プッシュすると会員TOPに反映されます。
- コード入力だけ行いたい場合は **master-login.html** から入り、コードを入力してマスターページへ進んでください。

## 更新するファイル

- **`member/data.js`** の `MANASEMI_DATA.calendar`（年間カレンダー）・`MANASEMI_DATA.youtube`（講座動画）・`MANASEMI_DATA.announcements`（直近の活動告知）を編集します。

---

## 年間カレンダー（calendar）

各要素は **month（月）・event（行事名）・date（日付）・venue（会場）** の4つです。

- **日付が決まったら**: 該当する行の `date: "—"` を `date: "5/10"` のように書き換える。
- **会場が決まったら**: 該当する行の `venue: "—"` を `venue: "〇〇会館"` のように書き換える。

### 例（第1講の日付 5/10、会場 〇〇会館 に決まった場合）

```javascript
{ month: "5月", event: "第1講", date: "5/10", venue: "〇〇会館" },
```

### 行事の追加・削除

- **追加**: `calendar` 配列に `{ month: "〇月", event: "行事名", date: "—", venue: "—" },` を追加する。
- **削除**: 不要な行のオブジェクトを削除する。

---

## 直近の講座動画（youtube）

各要素は **videoId（YouTubeの動画ID）・title（表示タイトル）・published（公開日表示）** の3つです。配列の**先頭**が会員TOPで一番上に表示されます。

- **動画を追加**: `youtube` 配列に `{ videoId: "動画ID", title: "タイトル", published: "2026/07/15" },` を追加する。videoId は YouTube の URL `?v=xxxxx` の xxxxx の部分。
- **既存の動画を修正**: 該当オブジェクトの `videoId`・`title`・`published` を書き換える。
- **動画を削除**: 不要なオブジェクトを削除する。

### 例（動画を1本追加）

```javascript
{ videoId: "xxxxxxxxxxx", title: "第3講 講座タイトル", published: "2026/07/15" },
```

---

## 直近の活動告知（announcements）

各要素は **date（日付）・title（タイトル）・body（本文）** の3つです。配列の**先頭**が会員TOPで一番上に表示されます。

- **新しい告知を追加**: 配列の先頭に `{ date: "2026/04/01", title: "タイトル", body: "本文" },` を追加する。
- **既存の告知を修正**: 該当オブジェクトの `date`・`title`・`body` を書き換える。
- **告知を削除**: 不要なオブジェクトを削除する。

### 例（告知を1件追加）

```javascript
{ date: "2026/04/01", title: "第2回全体例会のご案内", body: "〇月〇日に全体例会を開催します。会場は追ってご連絡します。" },
```

---

## 反映の流れ

1. **data.js** を編集して保存する。
2. 会員TOP（**index.html**）をブラウザで再読み込みする。
3. カレンダー・講座動画・告知が更新された表示になる。

Git でコミット・プッシュすると、本番サイトにも反映されます。
